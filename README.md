# Учебный ML-проект: прогноз риска сердечной недостаточности

Это учебный проект, посвящённый применению методов машинного обучения для классификации клинического случая по риску сердечной недостаточности.

В проекте реализована веб-страница на **Streamlit**, где пользователь может ввести признаки пациента и получить:
- вывод модели: относится ли случай к группе риска или нет;
- визуальную интерпретацию факторов, которые сильнее всего повлияли на результат;
- краткие медицинские пояснения по каждому значимому признаку.

Проект разработан в учебных целях и предназначен прежде всего для студентов-медиков, чтобы на практическом примере изучать:
- как клинические признаки используются в ML-моделях;
- как модель принимает решение;
- какие факторы могут усиливать или снижать прогнозируемый риск.

## Возможности проекта

- ввод клинических признаков пациента через удобный веб-интерфейс;
- автоматическая классификация случая по риску сердечной недостаточности;
- графическая интерпретация наиболее значимых признаков;
- справка по используемым медицинским параметрам.

## Технологии

- Python
- Pandas
- Scikit-learn
- Streamlit
- Altair
